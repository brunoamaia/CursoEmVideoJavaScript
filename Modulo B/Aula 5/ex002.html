Aula 5 - Variáveis

Para criar uma variável em JavaScript, devemos seguir as segiuntes regras:

  • Podem começar com letra, $ ou _.
  • Não podem começar com números.
  • É possível usar acentos e símbolos.
  • Não podem conter espaços.
  • Não podem ser palavras reservadas.


Tipos de dados

  1. number
    • Infinity
    • NaN
  2. string
  3. boolean
    • false, true
  4. null
  5. undefined
  6. object
    • Array
  7. function
