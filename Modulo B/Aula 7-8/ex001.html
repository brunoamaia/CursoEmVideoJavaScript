Aulas 7 e 8

Operadores

Aritméticos

  • + :  Soma (5+2=7)
  • - :  Subtração (5-2=3)
  • * :  Multiplicação (5*2=10)
  • / :  Divisão (5/2=2,5)
  • % :  Resto da divisão inteira (5%2=1)
  • ** :  Potenciação (5**2=25)

    • Precedência: ( )  >  **  >  *, /, %  >  +, -

Atribuição

  • n+=x  => n = n+x
  • n-=x  => n = n-x
  • n*=x  => n = n*x
  • n/=x  => n = n/x
  • n**=x  => n = n**x
  • n%=x  => n = n%x

Relacionais

  • 5>2  => true
  • 7<4  => false
  • 8>=8  => true
  • 9<=7  => false
  • 5=='5'  => true (verifica apenas o que foi armazenado)
  • 5==='5'  =>false (verifica o valor/string e os tipos)
  • 4!=4  => false

Lógicos

  • ! -> Negação
  • && -> Conjunção (e)
  • || -> Disjunção (ou)

Ternário

    ? :  (Funciona como um teste de erro)
    teste ? true : false  (Teste tal coisa, se for vedadeiro faça A, se for falso, faça B)
    Exemplo:  nota>6 ? 'Aprovado' : 'Reprovado'

Precedência Geral

  • Aritméticos: ( ), **, *, /, +, -
  • Atribuição: <, >, <=, >=, ==, !=
  • Negação: !
  • Conjunção: &&
  • Disjunção: ||
